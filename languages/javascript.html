Zastosowanie

Javascript jest najczęściej używanym językiem webowym, który służy zapewnieniu interaktywności stron oraz obsłudze zdarzeń, walidacji formularzy i budowaniu elementów nawigacyjnych. Można go także wykorzystywać do pisania pełnoprawnych aplikacji dzięki Windows Host Scripting. Ma mnóstwo frameworków jak Node.js pozwalający na interpretacje poprzez serwer, oraz JQuery - ogromna biblioteka ułatwiająca pisanie w tym języku i korzystanie z szeregu gotowych funkcji.

Node.js wieloplatformowym środowiskiem uruchomieniowym JavaScript służącym do wykonywania kodu po stronie serwera. Został wykorzystany o popularny silnik V8 - używany chociażby w Google Chrome, jednocześnie zapewniając bardzo szybkie wykonanie kodu. Pozwala na stworzenie własnego serwera HTTP, aby zaimplementować własny blog lub lekką aplikację internetową, jak i na inne usługi sieciowe. Równocześnie istnieje wiele poradników oraz przykładowych projektów, które pozwalają na przyjemną naukę oraz pisanie swoich pierwszych aplikacji. Jest to framework, który w ostatnich latach zyskał bardzo na popularności, a obecnie jest używany przez wielu gigantów, takich jak: Yahoo, Netflix, Medium, Uber czy PayPal.

Poziom skomplikowania

JavaScript jest bardzo łatwym językiem do rozpoczęcia zwłaszcza ze względu na to że jest wysoko poziomowy, oraz do rozpoczęcia pisania wystarczy tylko notatnik oraz przeglądarka - bez potrzeb kompilowania kodu. Jednak największym problemem jest fakt, że ten sam kod może działać inaczej w Chrome oraz w Firefoxie, co często sprawia wiele problemów początkującym programistom. Nauka JavaScripta jest dość łatwa, jednak pełne opanowanie tego języka jest bardzo ciężkie.

Skalowalność

Jako dynamicznie typowany język, JavaScript jest dość wolny, jest zbyt elastyczny żeby móc szybko wykonywać operacje. Dodatkowo jego prędkość zależy od użytego frameworka, Node.js bardzo go przyspiesza, jako że back end musi być zoptymalizowany pod względem prędkości.

Jako wadę można uznać trudność do debugowania niż większość obecnych języków. Dodatkowo im większa aplikacja tym trudniej ją poprawnie utrzymać, jednak nowo wprowadzany przez Microsoft statycznie pisany TypeScript ma zapobiec tym problemom.

Społeczność

JavaScript jest bardzo popularnym językiem, między innymi że często jest pierwszym językiem programowania używanym przez uczniów techników informatycznych, który jest wprowadzany zaraz po HTML oraz CSS. JavaScript jest obecnie na 8 miejscu w rankingu TIOBE co zapewnia mu bardzo wysoką popularność zwłaszcza przy fakcie, że zainteresowanie tym językiem wzrosło o 22.3% w ubiegłych latach.

JavaScript jest także najbardziej oznaczanym językiem na Githubie z łączną ilością projektów przekraczającą 3100 przy liczbie 5 gwiazdek. Jego popularność na portalu StackOverflow jest także niczego sobie, jest to najbardziej popularny język z liczbą pytań równą 1,302,415. Jest to także najbardziej popularna społeczność na portalu Meetup.com z 2,377,702 członkami oraz 4,583 spotkaniami na całym świecie.

Przyszłość języka

JavaScript wraz z rozwojem aplikacji będzie z pewnością rósł na popularności, dzięki temu, że jest wykorzystywany w praktycznie każdej aplikacji mobilnej oraz wraz z rozwojem coraz to nowszych frameworków poszerzających jego możliwości.
