Przyszłość

Python jest dynamicznym językiem programowania wysokiego poziomu opracowanym we wczesnych latach 90. przez Guido van Rossuma. Jego nazwa pochodzi od Monty Python'a, co miało oznaczać przyjemność oraz zabawę podczas korzystania z tego języka, bardzo ułatwiając naukę początkującym programistom. Charakteryzuje się łatwością pisania, czytelnym kodem, jak i bogatą biblioteką, która pozwala na jego wszechstronne zastosowanie. Jedną z jego największych zalet, jest fakt, że przy o wiele mniejszym nakładzie pracy możemy osiągnać bardzo przybliżony efekt jak przy użyciu innych języków.

Składnia

Poniższy przykład przedstawia aplikację generujacą silne hasła - o dowolnej ilości znaków wybranych przez użytkowników, oraz losową ilością wielkich, małych liter, cyfr oraz znaków specjalnych. 


            import random
            import string


            def main():
                print("[*] Welcome to Password Generator")
                print("[*] Remember that strong password should be longer than 8 characters!")
                length = input("[*] Enter how long would you like your password to be: ")
                if length.isdigit():
                    length = int(length)
                else:
                    print("[*] Please enter a number!")
                    return
                if length <= 3:
                    print("[*] It's not possible to generate the password. Please use longer one. ")
                else:
                    password_strong = False
                    while not password_strong:
                        password = generate(length)
                        password_strong = check(password)
                    print("[*] Your password has been generated: " + ''.join(password))

            def generate(length):
                password = ''.join(random.SystemRandom().choice(string.ascii_letters + string.punctuation + string.digits) for _ in range(length))
                return password

            def check(password):
                lower = sum(1 for letter in password if letter.islower())
                upper = sum(1 for letter in password if letter.isupper())
                digit = sum(1 for letter in password if letter.isnumeric())
                special = sum(1 for letter in password if letter in string.punctuation)

                if lower > 0 and upper > 0 and digit > 0 and special > 0:
                    return True
                else:
                    return False

            if __name__ == '__main__':
                main()
