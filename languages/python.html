Zastosowanie

Python jest językiem o bardzo szerokim zastosowaniu, pozwalającym na napisaniu dosłownie wszystkiego przy użyciu odpowiednich bibliotek lub frameworków, przez co bardzo wiele firm używa go w swoich aplikacjach. Django - back-end framework pozwalający na pisanie stron internetowych jest używany przez portal Instagram, tak samo jak YouTube oraz Reddit, które są napędzane przez ten język. BitBucket, który jest serwisem internetowym, obsługującym system kontroli wersji Merculiar oraz Git oraz pozwalajacym na wspólną pracę z kodem drużynie programistów. Został on napisany przy użyciu Django. Tak samo jak Merculiar - który w bardzo dużej mierze został napisany w Pythonie.

Python jest także używany do pisania aplikacji desktopowych. Klient Dropboxa na systemy Windows, MacOS oraz Linux został w dużej mierze napisane w Pythonie, tak samo jak Blender - program do obróbki grafiki 3D oraz znany wszystkim Gimp. A co z grami? Większość znanej gry Sid Meier's Civilization IV została również napisana w Pythonie co znacznie ułatwia graczom na zmianę zasad działania sztucznej inteligencji przeciwników oraz eksperymentowanie ze skryptami.

Poziom skomplikowania

Python został stworzony tak, aby był łatwy do zrozumienia i przyjazny początkującym. Kod Pythona można czytać jak zwykły tekst po angielsku. Nie trzeba kłaść dużego nacisku na składnie jak w innych językach. Jest to sporym ułatwieniem dla początkujących, przez co mogą skupić się na nauce właściwego programowania. Python pozwala także na dowolność, tzn nie ma sztywnych zasad tworzenia oprogramowania oraz wybacza więcej błędów.

Wydajność

Mimo swojej czytelności oraz faktu że często wymaga najmniejszej ilości linijek kodu do rozwiązania danego problemu, Python jest dość wolnym językiem. //Nie wiem więcej, nie ufam bestprogramminglanguage

Popularność

Popularność Pythona rośnie z każdym rokiem, co również ciągnie za sobą coraz większą liczbę zwolenników, którzy gotowi są pomagać innym członkom tej społeczności. Na StackOverflow jest to 5 największa społeczność, z liczbą ponad 600 000 pytań, rozwiązujących praktycznie każdy problem, który mógł napotkać początkujący, a nawet zaawansowany programista. Jest on także 4 najbardziej popularnym językiem na serwisie GitHub, tworząc pokaźną bibliotekę kodu, które można wykorzystać jako biblioteki do przyszłych aplikacji lub jako wzory gotowych aplikacji - do późniejszego odtworzenia.

Natomiast jeśli ktoś woli uczyć się na 'żywo', rozwiązując problemy wraz z innymi przy kawie i ciastku to społeczność na Meetup.com także nie rozczarowuje - łącznie ponad 1,033,429 członków na całym świecie oraz 1729 spotkań. Z takim wyborem każdy na pewno odnajdzie coś dla siebie.

Przyszłość języka

Python jako projekt otwartoźródłowy jest cały czas aktywnie dopracowywany, by sprostać coraz to nowszym wymaganiom. Rozwój uczenia maszynowego (machine learning), które jest witalną częścią rozwoju sztucznej inteligencji, jak i zarówno rosnąca ilość big data wróżą świetlaną przyszłość dla tego języka.
