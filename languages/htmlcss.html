Po co stworzono HTML i CSS?

HTML (ang. HyperText Markup Language) - Jest to język znaczników za pomocą, którego można tworzyć strony internetowe lub aplikacje internetowe. Razem z CSS(Cascading Style Sheets - Kaskadowe arkusze stylów), które umożliwiają modyfikowanie styli i pozycji elementów na stronie oraz JavaScript składają się na podwaliny technologii dla World Wide Web (światowej rozległej sieci internetowej).

HTML

Znaczniki HTML są budulcem każdej strony napisanej w tym języku, umożliwiają między innymi wstawianie obrazków, paragrafów, nagłówków, dzielenie strony na sekcje. Znaczniki zapisywane z pomocą nawiasów (np. <img>, który pozwala na dodanie obrazka do strony) wstawiają treść na stronę, przeglądarki nie wyświetlają ich, ale użwywają do zinterpretowania zawartości strony.

CSS

CSS jest używany do opisania prezentacji (wyświetlania) strony napisanej w HTML, został stworzony w celu odseparowania struktury dokumentu od formy jego prezentacji. Separacja ta zwiększa zakres dostępności witryny, zmniejsza zawiłość dokumentu, ułatwia wprowadzanie zmian w strukturze dokumentu. Do znacznego ułatwienia i przyspieszenia edytowania stylów używane są preprocesory oraz frameworki. Preprocesory takie jak LESS oraz SASS wprowadzają elementy znane z języków programowania czyli między innymi funkcje, instrukcje warunkowe, pętle. Oczywiście taki kod wygenerowany przez preprocesor nie mógłby zostać poprawnie wczytany i zinterpretowany przez przeglądarkę, zatem trzeba go najpierw zamienić na wynikowy kod CSS. Framework natomiast jest to zbiór gotowych stylów, który ułatwia i znacznie przyspiesza budowanie stron, jednym z bardziej popularnych jest Bootstrap (dzięki niemu powstał jaki-jezyk-programowania.pl).
