C#

OPISY MOGĄ ZAWIERAĆ BŁEDNE INFORMACJE
Aktualizacja: 25.01.2017

Zastosowanie

C# jest wysokopoziomowym, zorientowanym obiektowo językiem programowania ogólnego przeznaczenia, który jest odpowiedzią Microsoftu na Jave. C# jest ścisłe zintegrowany z platformą .NET, która jest zarówno frameworkiem jak i środowiskiem uruchomieniowym.

C# najczęściej stosowany jest do pisania aplikacji na systemy Windows. Ale to nie wszystko. C# pozwala także na tworzenie aplikacji webowych po stronie serwera za pomocą frameworka ASP.NET.

Dzięki narzędziom takim jak Xamarin, możliwe jest też programowanie wieloplatformowych aplikacji mobilnych.

C# używany jest też w popularnym silniku Unity, za pomocą, którego można tworzyć gry na PC, konsole, urządzenia mobilne czy strony internetowe

C# napędza m.in. stronę StackOverflow czy gry takie jak Terraria, Magicka, Bastion.

Poziom skomplikowania

Dla osób które miały konakt z takimi językami jak C, C++ lub Java składnia będzie bardzo znajoma. Składnia C# jest uproszczoną wersją tej z C++, więc jeżeli programowanie obiektowe cię nie przeraża i chcesz pisać aplikacje głównie na system Windows to C# może być dobrym językiem na start.

Wydajność

C# podobnie jak Java posiada kompilator JIT(Just in Time), jest także językiem silnie typowanym co oznacza że program zwróci błąd i zapobiegnie kompilacji, jeżeli przekazane dane nie będą się zgadzać z typem danych jakie zadeklarowaliśmy dla zmiennej, w przeciwieństwie do języka słabo typowanego, który nie wygeneruje błędu i zwróci nieprzewidywany wynik. Kompilacja JIT znana także jako dynamiczna translacja, w której program jest kompilowany w trakcie jego działania, czyli program jest przetwarzany i jednocześnie wykonywany bez czekania na skompilowanie całego kodu, JIT bada także, które części programu bardziej opłaca się skompilować i w takiej postaci zostawić od tych gdzie translacja dynamiczna jest skuteczniejsza. Cechy te sprawiają że C# osiąga szybkość porównywalną z Javą.

Popularność

C# zajmuje 4 miejsce pod względem popularności według TIOBE index z 2% stratą do C++. Jest 3 na Stack Overflow z ponad 1 050 000 zapytaniami, jednak w społeczności Github nie cieszy się dużą popularnością, zajmując 10 pozycję z ponad 300 000 repozytoriami. Meetup.com podpowiada nam, że członków na całym świecie jest 259 615 z 691 spotkaniami. Duża popularność silnika Unity (który może się pochwalić tym, że 34% najpopularniejszych gier mobilnych powstało w tym silniku oraz notowaniem ciągłych zysków w sektorze gier mobilnych i desktopowych) wpływa także pozytywnie na rozgłos C#.

Przyszłość języka

Jako że Microsoft wydał C# aby tworzyć aplikacje na Windowsa nie zamierza kończyć dla niego wsparcia, w niedalekiej przyszłości planuje wydać wersje 7. Szybki rozwój Unity powoduje, że programiści C# mają ogromną szansę na osiągnięcie sukcesu w tworzeniu gier.
