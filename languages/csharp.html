C#

Dane aktualne na dzień 25.01.2017

Zastosowanie

C# ("C sharp", zamiennie z C# stosuje się C♯) jest wysoko poziomowym, zorientowanym obiektowo językiem programowania, który jest odpowiedzią Microsoftu na Jave. Pozwala on na zbudowanie rozmaitych programów poprzez platformę .NET Framework takich jak aplikacje usług internetowych XML, aplikacje klient-serwer, aplikacje baz danych, oprogramowanie i wiele więcej. Jednak C# najbardziej rozwija się pod kątem pisania aplikacji na Windows, dzięki czemu język ten jest najbardziej popularny wśród programistów piszących programy na systemy Microsoftu.

C# napędza między innymi silnik Unity, portal StackOverflow, Evernote czy gry takie jak Terraria, Magicka, Bastion. Platforma niezbędna do pisania w C# czyli .NET Framework obejmuje środowisko uruchomieniowe (które kompiluje kod źródłowy) oraz biblioteki klas (klasa pozwalana definiowanie i modyfikację obiektów, a gotowe bibliteki znacznie przyspieszają proces tworzenia programu). C# bez tej platformy nie mógłby istnieć. .NET Framework obsługuje także C++/CLI, F#, J#, Delphi 8 dla .NET, Visual Basic .NET.

Poziom skomplikowania

Dla osób które miały konakt z takimi językami jak C, C++ lub Java składnia będzie bardzo znajoma, a programiści którzy mieli kontakt z Javą nie będą mieli problemów z pisaniem programów zorientowanych obiektowo. Składnia C# jest uproszczoną wersją tej z C++, więc jeżeli programowanie obiektowe cię nie przeraża i chcesz pisać aplikacje głównie na system Windows to C# może być dobrym językiem na start.

Wydajność

C# podobnie jak Java posiada kompilator JIT(Just in Time), jest także językiem silnie typowanym co oznacza że program zwróci błąd i zapobiegnie kompilacji, jeżeli przekazane dane nie będą się zgadzać z typem danych jakie zadeklarowaliśmy dla zmiennej, w przeciwieństwie do języka słabo typowanego, który nie wygeneruje błędu i zwróci nieprzewidywany wynik. Kompilacja JIT znana także jako dynamiczna translacja, w której program jest kompilowany w trakcie jego działania, czyli program jest przetwarzany i jednocześnie wykonywany bez czekania na skompilowanie całego kodu, JIT bada także, które części programu bardziej opłaca się skompilować i w takiej postaci zostawić od tych gdzie translacja dynamiczna jest skuteczniejsza. Cechy te sprawiają że C# osiąga szybkość porównywalną z Javą.

Popularność

C# zajmuje 4 miejsce pod względem popularności według TIOBE index z 2% stratą do C++. Jest 3 na Stack Overflow z ponad 1 050 000 zapytaniami, jednak w społeczności Github nie cieszy się dużą popularnością, zajmując 10 pozycję z ponad 300 000 repozytoriami. Meetup.com podpowiada nam, że członków na całym świecie jest 259 615 z 691 spotkaniami. Duża popularność silnika Unity (który może się pochwalić tym, że 34% najpopularniejszych gier mobilnych powstało w tym silniku oraz notowaniem ciągłych zysków w sektorze gier mobilnych i desktopowych) wpływa także pozytywnie na rozgłos C#.

Zarobki

Według wynagrodzenia.pl miesięczne wynagrodzenie całkowite (mediana) programisty .NET (w którego pakiet wchodzi język C#) stanowisku wynosi 6 000 PLN brutto. Co drugi programista .net otrzymuje pensję od 4 300 PLN do 8 176 PLN. 25% najgorzej wynagradzanych programistów .net zarabia poniżej 4 300 PLN brutto. Na zarobki powyżej 8 176 PLN brutto może liczyć grupa 25% najlepiej opłacanych programistów .net. Natomiast według Payscale.com zarobki w Anglii wynoszą 2 698.75£.

Przyszłość języka

Jako że Microsoft wydał C# aby tworzyć aplikacje na Windowsa nie zamierza kończyć dla niego wsparcia, w niedalekiej przyszłości planuje wydać wersje 7. Szybki rozwój Unity powoduje, że programiści C# mają ogromną szansę na osiągnięcie sukcesu w tworzeniu gier.
