Przeczytaj o ASP.NET na oficjalnej stronie

ASP.NET to wieloplatformowy framework open-source do budowania aplikacji webowych, oparty o model MVC. ASP.NET wspomaga tworzenie dynamicznych strony internetowe w języku C# (I każdym innym działającym w platformie .NET) zbudowane na HTML5, CSS i Javascript.
